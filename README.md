# RATIONAL_LIB

Библиотека для работы с рациональными числами на языке C.

## Индивидуальное задание

Разработать библиотеку для работы с рациональными числами. Реализованы:

-  структура rational_t
-  нормализация дроби (сокращение на НОД)
-  сложение, вычитание, умножение, деление
-  сравнение двух чисел
-  преобразование в double
-  C-тесты
-  Python-тесты через ctypes
